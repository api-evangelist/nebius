Financial institutions are adopting transaction foundation models—transformer architectures trained on proprietary transaction data for tasks like fraud detection and credit scoring. Revolut's PRAGMA model, trained on 40 billion events using 64 NVIDIA H100 GPUs on Nebius AI Cloud, demonstrates production-scale implementation with a 21% precision improvement on fraud detection. Nebius AI Cloud provides the infrastructure needed for data preparation, training, and inference deployment of these models.
