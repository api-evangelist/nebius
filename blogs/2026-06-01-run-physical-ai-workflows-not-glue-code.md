Nebius has released the Physical AI Workbench, an open-source platform that streamlines physical AI development by pre-integrating tools like NVIDIA Cosmos 3, Isaac Sim, and Isaac GR00T into composable building blocks. Rather than requiring teams to spend weeks stitching together separate containers and tools, the Workbench provides a unified interface where agents can orchestrate entire workflows—from synthetic data generation and simulation to training and evaluation—using natural language instructions. The closed-loop sim-to-real pipeline enables continuous model improvement through managed infrastructure on Nebius AI Cloud.
